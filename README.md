# Scentura Production

Einfaches Parfümproduktionssystem mit Node.js (v22.3), Express und REST-API.  
Frontend über statische HTML-Seiten, Backend mit modularer Struktur und JSON-Datenhaltung.

---

```bash
# Dependencies installieren
npm install

# Server starten
npm run dev

# Server laueft anschliessend unter:
